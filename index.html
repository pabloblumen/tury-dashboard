Conectar Google Sheets

Pegá la URL de tu Web App de Apps Script para conectar el dashboard en tiempo real con tus datos de pedidos.

1
En tu Google Sheet: Extensiones → Apps Script, pegá el código del archivo TURY_AppsScript.gs
2
Hacé clic en Implementar → Nueva implementación, tipo: Aplicación web, acceso: Cualquier persona
3
Copiá la URL de la aplicación web que te genera Google y pegala abajo
No se pudo conectar. Verificá la URL e intentá de nuevo.
